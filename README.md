# Sarthi - Your Career Guide

Sarthi is a comprehensive AI-powered career guidance platform designed to help students discover their ideal career streams, find scholarships, explore internships, and practice with mock tests.

## Repository Structure
- **[/client](file:///c:/Users/saksh/OneDrive/Desktop/sarthi-mern(final)/client)**: React + Vite + TypeScript frontend (PWA).
- **[/server](file:///c:/Users/saksh/OneDrive/Desktop/sarthi-mern(final)/server)**: Node.js + Express + MongoDB backend API.

## Quick Start

### 1. Backend Setup
```bash
cd server
npm install
npm run dev
```

### 2. Frontend Setup
```bash
cd client
npm install
npm run dev
```

## Features
- 🚀 **PWA Enabled**: Installable on Android, iOS, and Desktop.
- 🧠 **AI Career Quiz**: Deep assessment for stream selection.
- 🎓 **Scholarship Finder**: Latest government and private opportunities.
- 💼 **Internship Portal**: Domain-specific listings.
- 🤖 **AI Chatbot**: 24/7 career mentoring.
