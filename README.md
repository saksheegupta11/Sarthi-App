# 🎓 Sarthi - Your Ultimate Career Guide

Sarthi is an advanced AI-powered career guidance platform designed to empower students with data-driven insights. It helps students navigate their career paths through personalized assessments, scholarship discovery, internship opportunities, and real-time AI mentoring.

---

## 🚀 Key Features

- 🧠 **AI Career Assessment**: Dynamic quizzes that analyze your interests and skills to suggest the best career streams.
- 🏫 **College Finder**: Explore a comprehensive directory of Indian institutions (IITs, NITs, IIITs, etc.) with real-time data.
- 🎓 **Scholarship Discovery**: Stay updated with the latest government and private scholarship opportunities.
- 💼 **Internship Portal**: Find domain-specific internships to jumpstart your professional journey.
- 🤖 **AI Career Mentor**: 24/7 chatbot powered by Google Gemini for instant career advice and query resolution.
- 📱 **PWA Support**: Installable application for a seamless mobile experience on Android, iOS, and Desktop.
- 💾 **Personalized Dashboard**: Save colleges, scholarships, and internships to track your progress.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: React.js with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Hooks & Context API
- **Deployment**: Vercel

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose)
- **AI Integration**: Google Generative AI (Gemini)
- **Deployment**: Vercel / Render

## 📂 Project Structure

- **[/client](./client)**: React frontend application (PWA).

## ⚙️ Quick Start

### 1. Prerequisites
- Node.js (v18+)
- MongoDB (Local or Atlas)
- API Keys (Google Gemini, etc.)

### 2. Backend Setup
```bash
cd server
npm install
# Create a .env file based on .env.example
npm run dev
```

### 3. Frontend Setup
```bash
cd client
npm install
# Create a .env file based on .env.example
npm run dev
```
